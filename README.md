# 🎮 English Everyday — เกมฝึกภาษาอังกฤษทุกวัน

เกมฝึกภาษาอังกฤษในชีวิตประจำวัน เล่นได้ทันทีโดยไม่ต้องติดตั้งอะไรและไม่ต้องต่ออินเทอร์เน็ต

## วิธีเปิดเล่น

**ดับเบิลคลิกไฟล์ `index.html`** — แค่นั้นเลย! เกมจะเปิดในเบราว์เซอร์
(แนะนำ Microsoft Edge หรือ Google Chrome เพื่อให้เสียงอ่านภาษาอังกฤษทำงานสมบูรณ์)

เคล็ดลับ: คลิกขวาที่ `index.html` → Send to → Desktop เพื่อสร้างทางลัดไว้เปิดเล่นทุกวัน

## มีอะไรในเกมบ้าง

- 🎯 **แบบทดสอบวัดระดับ** — เปิดครั้งแรกจะมีแบบทดสอบ 15 ข้อ เพื่อจัดระดับความยากให้เหมาะกับคุณ (เริ่มต้น / กลาง / ก้าวหน้า) เปลี่ยนระดับทีหลังได้ที่หน้าโปรไฟล์
- 🗣️ **บทสนทนาสถานการณ์จริง** — เล่นเป็นตัวละคร 10 สถานการณ์ เช่น สั่งกาแฟ ถามทาง เช็คอินโรงแรม ไปหาหมอ
- 🧠 **คำศัพท์ประจำวัน** — แฟลชการ์ด 120 คำ 8 หมวด + ควิซที่วนคำที่ตอบผิดกลับมาให้ฝึกบ่อยขึ้น
- ✏️ **เติมคำในประโยค** — แกรมมาร์ที่คนไทยพลาดบ่อย พร้อมเฉลยอธิบายภาษาไทย
- 🎧 **ฟัง-พิมพ์ตามเสียง** — ฝึกหูกับเสียงอ่านภาษาอังกฤษ ปรับความเร็วช้าได้ ขอคำใบ้ได้
- 📅 **ภารกิจรายวัน** — ทุกวันมีภารกิจใหม่ 3 อย่าง ทำครบรับโบนัส XP ก้อนใหญ่
- ⭐ **XP · เลเวล · ความสำเร็จ 12 เหรียญ** — เห็นพัฒนาการตัวเองชัดเจน

## ข้อมูลการเล่น

ความคืบหน้าทั้งหมด (XP, เหรียญ, สถิติ) ถูกเก็บไว้ในเบราว์เซอร์เครื่องนี้ (localStorage)
ถ้าอยากเริ่มใหม่ทั้งหมด ไปที่ **โปรไฟล์ → ล้างข้อมูลทั้งหมด**

Have fun and keep practicing every day! 💪
